# Continuum AI

Continuum AI is an executive meeting intelligence platform that converts meeting transcripts into structured executive insights.

## Features

- Meeting transcript upload
- AI-powered analysis using Gemini
- Key Decisions extraction
- Dependency identification
- Risk detection
- Recommended actions
- Decision Continuity Health Score
- PDF report export

## Tech Stack

- HTML
- CSS
- JavaScript
- Gemini API
- Vercel Serverless Functions

## Live Demo

https://continuum-neon.vercel.app

## Installation

Clone the repository:

```bash
git clone https://github.com/premg326/continuum-ai.git
```

Install dependencies:

```bash
npm install
```

Create a .env file:

```env
GEMINI_API_KEY=your_api_key
```

Run locally and deploy with Vercel.
